Osnove lista

Definicija

Lista je skupina elemenata koji su pohranjeni u jednoj varijabli. Listu definiramo unutar uglatih zagrada ([ i ]), a svaki se element odvaja zarezom. Moguće je i imati listu koja je prazna ([]).

Primer

l1 = []
l2 = [1, 2, 3]
l3 = ["a", "b", "c"]

Duljina lista

Duljina liste je broj njezinih elemenata. Npr. duljina liste l2 je 3 jer sadrži tri elementa. Duljinu liste možemo dobiti koristeći funkciju len.

print(len(l1))
print(len(l2))
print(len(l3))

Ispis liste

Elemente liste možemo ispisati stavljajući listu u printfunkciju. Ovaj način se ne priznaje na natjecanjima. 

print(l1)
print(l2)
print(l3)
