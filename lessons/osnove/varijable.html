Varijable

Osnove

Varijable imaju svoje ime i vrijednost. Njihova funkcija je držati neku vrijednost koju kasnije možemo koristiti znamo li njeno ime. Deklariramo („napravimo“) varijable tako što napišemo ime varijable, jednakost ( = ) i vrijednost. Na primjer:

x = 0

Tipovi varijabli

Varijable mogu imati različite tipove vrijednosti koji kažu kako pohraniti podatke. Najosnovniji su:

a = 1
b = 0
c = -1
a = 1.2
b = -1.2
c = 2.0
a = "abc"
a = True
b = False

Ispis vrijednosti varijabli

Možemo ispisati vrijednosti varijabli tako što napišemo ime varijable u print funkciju bez navodnika:

a = 10
print(a)

Na ovaj način možemo ispisati kombinaciju stringa i vrijednosti varijable:

a = 10
b = 20
c = 30
d = 40
print("Vrijednost varijable a:", a)
print("Vrijednost varijable b:", b)
print("Vrijednost varijable c:", c)
print("Vrijednost varijable d:", d)

Stavimo f prije navodnika kako bismo napravili f-string. Tamo gdje stavimo vitičaste zagrade { i } stavit će vrijednost varijable čije smo ime stavili. To je lakši način za ispisivanje s varijablama:

print(f"Vrijednost varijable a je {a}!")
print(f"Vrijednost varijable b je {b}!")
print(f"Vrijednost varijable c je {c}!")
print(f"Vrijednost varijable d je {d}!")

Promjena vrijednosti varijable

Možemo i promjeniti vrijednost varijable:

a = 10
print(a)
a = 20
print(a)
