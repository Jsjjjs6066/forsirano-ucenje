Varijable

Varijable imaju svoje ime i vrijednost. U Pythonu možemo koristiti varijable kako bismo pohranili vrijednosti. Deklariramo varijable tako što napišemo ime varijable, jednakost ( = ) i vrijednost. Na primjer:

x = 0

Ime varijable je "x", a vrijednost 0


Varijable mogu imati različite tipove vrijednosti:

a = 1

Cijeli broj (int)

b = 1.2

Decimalni broj (float)

c = "abc"

Skupina znakova (svaki znak može biti broj, slovo, "."...) (string)

d = True

Točno ili netočno (boolean)


Možemo ispisati vrijednosti varijabli tako što napišemo ime varijable u print funkciju bez navodnika:

a = 10
print(a)

Na ovaj način možemo ispisati kombinaciju imena varijable i vrijednosti:

print("a:", a)
print("b:", b)
print("c:", c)
print("d:", d)

Lakši način za ispisivanje s varijablama:

print(f"\nVrijednost varijable a je {a}!")
print(f"Vrijednost varijable b je {b}!")
print(f"Vrijednost varijable c je {c}!")
print(f"Vrijednost varijable d je {d}!")

Stavimo f prije navodnika kako bismo napravili f-string. Tamo gdje stavimo vitičaste zagrade { i } stavit će vrijednost varijable čije smo ime stavili.


Možemo i promjeniti vrijednost varijable:

a = 10

print(a)

a = 20

print(a)
