If naredba

Definirajmo varijablu s koja će spremiti unos korisnika:

a = int(input())
if a % 2 == 0:
    print("Paran je")

elif = else if, tj. "ako nije prethodno bilo točno ali ovo je".

elif a % 2 == 1:
    print("Neparan je")
if a % 3 == 0:
    print("Dijeljiv s 3")

else = ako nije zadovoljen nijedan prethodni uvjet

else:
    print("Nije dijeljiv s 3")
