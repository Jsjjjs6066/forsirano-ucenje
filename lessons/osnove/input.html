Input

Jednostavni primjer s imenom

U ovom primjeru korisnik će unijeti svoje ime, a program će ga ispisati.

ime = input("Koje ti je ime? ")
print("Tvoje ime je " + ime)

Ovo možemo koristiti u aplikacijama za korisnike ali u natjecateljskom programiranju ne smijemo staviti string u input funkciju. Ovako bismo to napravili u natjecateljskom programiranju:

ime = input()
print("Tvoje ime je " + ime)

Loš primjer s brojevima (daje error)

s = input()
print(s + 1)

Dobar primjer s brojevima

n = int(input())
String Int 
print(n + 1)
print(str(n) + " " + str(n + 1))
String String String 
print(n + " " + (n + 1))
